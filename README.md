# 💼 LinkedIn Feed UI Clone (React)

A simple **LinkedIn Feed UI Clone** built with **React.js** to practice component-based development, props, and reusable UI design. This project recreates the basic structure of a LinkedIn home feed using static data and custom styling.

---

## ✨ Features

- Responsive LinkedIn-inspired UI
- Reusable React Components
- Dynamic rendering using Props
- Post card layout
- Search/Create Post section
- Modern CSS styling
- SVG icons integration
- Clean component structure

---

## 🛠️ Built With

- React.js
- JavaScript (ES6+)
- JSX
- CSS3
- Vite
- SVG Icons
